# Hellobooks AI Assistant

This project is a simple AI assistant for answering accounting questions.

It uses Retrieval Augmented Generation (RAG).

## Technologies

Python
LangChain
FAISS
HuggingFace Embeddings

## Setup

Install dependencies

pip install -r requirements.txt

Build vector database

python build_vector_db.py

Run assistant

python app.py

## Example Questions

What is bookkeeping?
Explain profit and loss
What is balance sheet?
